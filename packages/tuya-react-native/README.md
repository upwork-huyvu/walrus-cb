# @cool-bath/tuya-react-native

React Native TurboModule wrapping Tuya Smart Life App SDK (Wi-Fi + BLE) for the ice-bath app

## Installation


```sh
npm install @cool-bath/tuya-react-native
```


## Usage


```js
import { multiply } from '@cool-bath/tuya-react-native';

// ...

const result = multiply(3, 7);
```
